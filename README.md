
# School Bazaar

School bazaar for SPSUL.cz. Practions from douda

## Run Locally

Clone the project

```bash
git clone https://github.com/HuricanSpeed/school-bazar.git
cd school-bazar
```
Go to schoolbazar or backend

```bash
cd schoolbazar
```
or
```bash
cd backend
```

Install dependencies

```bash
npm install
```

Start the server
```bash
ng s -o
```

or for backend
```bash
npm run start
```

Example of .ENV File
```bash
DB_HOST='localhost'
DB_USERNAME='root'
DB_PASS=''
DB_NAME='schoolbazar'
```
